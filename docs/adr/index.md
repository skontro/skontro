# Architecture Decision Records

This directory contains the architecture decisions made for Skontro, recorded using the [Michael Nygard template](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions). New ADRs are required for changes touching the data model, authentication, deployment, dependencies, or public API.
