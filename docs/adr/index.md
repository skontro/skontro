# Architecture Decision Records

This directory contains the architecture decisions made for Skontro, recorded using the [Michael Nygard template](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions). New ADRs are required for changes touching the data model, authentication, deployment, dependencies, or public API.

|---|---|---|---|

| [0002](0002-multi-tenancy.md) | Multi-tenancy via single database with tenant_id discriminator | Accepted | 2026-05-27 |
| [0003](0003-commit-conventions.md) | Adopt Conventional Commits 1.0.0 with commitlint enforcement | Accepted | 2026-05-28 |
| [0004](0004-documentation-framework.md) | Adopt Diátaxis-structured documentation powered by MkDocs Material | Accepted | 2026-05-28 |
